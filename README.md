# EAU Guidelines Quiz

Client-side MCQ quiz app for [EAU Guidelines](https://uroweb.org/guidelines), hosted on GitHub Pages.

**Stack:** Vanilla JS/CSS · No build step · No backend · Anki export via sql.js (CDN)

---

## How it works

```
uroweb.org → scrape HTML → JSON data
                          → MiniMax-M2.7 (LOCAL) → MCQ JSON
                          → web UI (quiz + export)
```

**CI only scrapes** — no LLM calls, no API keys. Generation runs on your machine.

---

## Quick start

### 1. Scrape all guidelines

```bash
pip install --quiet requests  # or just use stdlib (already in scripts)

python scripts/scrape_guideline_index.py        # gets list of all guidelines
python scripts/scrape_chapter_text.py --all      # scrapes every chapter
```

### 2. Generate MCQs (LOCAL — needs API key)

```bash
export MINIMAX_API_KEY=your_key
python scripts/generate_quiz_json.py --all
```

### 3. Serve locally

```bash
cd web
python -m http.server 8000
# open http://localhost:8000
```

## Anki export

Select guidelines in the web UI → "Export Anki Deck" → downloads a `.apkg` file.
Uses [sql.js](https://sql.js.org/) (CDN) to build a SQLite `.apkg` readable by Anki Desktop and AnkiMobile.

---

## Deploy

1. Push code to `jhyrachy/eau-quiz`
2. Settings → Pages → Source: `main`, folder: `/web`
3. CI runs on push to `main` — scrapes and commits new data

---

## Updating questions

Guidelines don't change often (~yearly). To regenerate:

```bash
export MINIMAX_API_KEY=your_key
python scripts/generate_quiz_json.py --all
git add data/
git commit -m "update questions"
git push
```
